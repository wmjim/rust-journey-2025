## 项目结构

```bash
rust-journey-2025/               # 主项目目录
├── Cargo.toml                   # 项目配置文件
├── src/                         # 主源码目录
│   ├── main.rs                  # 主入口文件
│   ├── lib.rs                   # 库文件
│   ├── day1_rust_foundation/    # Day1 代码
│   │   ├── mod.rs               # Day1 模块声明
│   │   ├── variables.rs         # Day1 变量练习实现
│   │   ├── functions.rs         # Day1 函数练习实现
│   │   ├── control_flow.rs      # Day1 控制流练习实现
│   │   └── match.rs             # Day1 match 语法实现
├── tests/                       # 集成测试目录
│   ├── day1_test.rs             # Day1 测试
└── benches/                     # 基准测试目录（可选）
```
